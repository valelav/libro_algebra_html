Capoverso

Sia \(V\) uno spazio vettoriale di dimensione \(n\) su un campo \(K\) e sia \(\theta\) un endomorfismo di \(V\text{.}\) Esistono allora polinomi irriducibili monici \(p_{1}\text{,}\) \(p_{2}\text{,}\) \(\ldots\text{,}\) \(p_{t}\) (non necessariamente distinti) e interi positivi \(\alpha_{1}\text{,}\) \(\alpha_{2}\text{,}\) \(\ldots\text{,}\) \(\alpha_{t}\) e una base di \(V\) rispetto a cui \(\theta\) si rappresenta con la matrice diagonale a blocchi

\begin{equation*} C\coloneqq \begin{pmatrix} C_{1} \amp 0 \amp \dots \amp 0 \\ 0 \amp C_{2} \amp \dots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp \dots \amp C_{t} \end{pmatrix} \end{equation*}

dove ciascun \(C_{i}\) è la matrice compagna del polinomio \(p_{i}^{\alpha_{i}}\text{.}\) La matrice \(C\) è univocamente determinata a meno dell'ordinamento dei blocchi \(C_{i}\text{.}\)
