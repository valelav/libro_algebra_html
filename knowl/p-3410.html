Consideriamo le matrici a coefficienti reali

\begin{equation*} A\coloneqq \begin{pmatrix} 0 \amp 1 \amp 0 \amp 0 \\ -1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp -1 \amp 0 \end{pmatrix} \end{equation*}

e

\begin{equation*} B\coloneqq \begin{pmatrix} 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ -1 \amp 0 \amp -2 \amp 0 \end{pmatrix}\text{.} \end{equation*}

La matrice \(A\) è una matrice diagonale a \(2\) blocchi di ordine \(2\text{:}\) ciascun blocco è la matrice compagna del polinomio irriducibile \(x^{2}+1\text{.}\) La matrice \(B\) è la matrice compagna del polinomio \((x^{2}+1)^{2}\text{.}\) Entrambe le matrici sono già in forma canonica (in questo caso particolare la forma canonica razionale e quella primaria coincidono) e hanno lo stesso polinomio caratteristico \((x^{2}+1)^{2}\text{,}\) tuttavia non sono simili tra loro.
