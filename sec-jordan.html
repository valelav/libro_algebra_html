Paragrafo 27.4 Forma canonica di Jordan

Proposizione 27.4.1.

Sia \(V\) uno spazio vettoriale di dimensione finita \(n\) su un campo \(K\) e sia \(\theta\) un endomorfismo di \(V\text{.}\) Supponiamo che \(V\text{,}\) con la struttura di \(K[x]\)-modulo indotta da \(\theta\text{,}\) sia ciclico di generatore \(\vect{v}\) e che l'annullatore di \(\vect{v}\) sia generato dal polinomio \((x-\lambda)^{n}\text{.}\) I vettori \(\vect{v}_{1}\coloneqq \vect{v}\text{,}\) \(\vect{v}_{2}\coloneqq \vect{v}(\theta-\lambda\Id)\text{,}\) \(\dots\text{,}\) \(\vect{v}_{n}\coloneqq\vect{v}(\theta-\lambda\Id)^{n-1}\) formano allora una base di \(V\) rispetto a cui \(\theta\) si rappresenta con la matrice

\begin{equation*} J_{n}(\lambda)\coloneqq \begin{pmatrix} \lambda \amp 1 \amp 0 \amp \dots \amp \dots \amp 0\\ 0 \amp \lambda \amp 1 \amp \dots \amp \dots \amp 0\\ \vdots \amp \vdots \amp \ddots \amp \ddots \amp \dots \amp \vdots\\ 0 \amp \dots \amp \dots \amp \lambda \amp 1 \amp 0 \\ 0 \amp \dots \amp \dots \amp 0 \amp \lambda \amp 1\\ 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp \lambda \end{pmatrix} \end{equation*}

detta blocco di Jordan 1  di ordine \(n\) relativo all'autovalore \(\lambda\text{.}\) Il polinomio caratteristico \(\det(x\Idmatrix-J_{n}(\lambda))\) della matrice \(J_{n}(\lambda)\) è esattamente \((x-\lambda)^{n}\text{.}\) L'endomorfismo \(\theta\) ha come unico autovalore \(\lambda\) di molteplicità algebrica \(n\) e molteplicità geometrica \(1\text{.}\)

Dimostrazione.

Mostriamo che gli \(n\) vettori \(\vect{v}_{1}\text{,}\) \(\dots\text{,}\) \(\vect{v}_{n}\) sono linearmente indipendenti e sono quindi una base per \(V\text{,}\) che sappiamo avere dimensione uguale al grado del polinomio ordine di \(\vect{v}\text{.}\) Siano allora \(b_{1}\text{,}\) \(\dots\text{,}\) \(b_{n}\) elementi di \(K\) tali che \(\vect{v}_{1}b_{1}+\dots+\vect{v}_{n}b_{n}=\vect{0}\text{,}\) vale a dire \(\vect{v}b_{1}+\vect{v}(\theta-\lambda\Id)b_{2}+\dots+\vect{v}(\theta-\lambda\Id)^{n-1}b_{n}=\vect{0}\text{,}\) cioè \(\vect{v}(b_{1}+b_{2}(x-\lambda)+\dots+b_{n}(x-\lambda)^{n-1})=\vect{0}\text{.}\) Poiché l'annullatore di \(\vect{v}\) è generato da un polinomio di grado \(n\) ciò comporta che \(b_{1}+b_{2}(x-\lambda)+\dots+b_{n}(x-\lambda)^{n-1}\) è uguale a \(0\text{.}\) Se espandiamo questo polinomio notiamo che il coefficiente di \(x^{n-1}\) è \(b_{n}\) e, dunque, \(b_{n}=0\text{.}\) Una volta posto \(b_{n}=0\text{,}\) il coefficiente di \(x_{n-2}\) è \(b_{n-1}\) e, dunque, \(b_{n-1}=0\) e così via.

Mostriamo ora che la matrice rappresentativa di \(\theta\) rispetto a questa base è la matrice \(J_{n}(\lambda)\) data in enunciato. Per \(1\le j\le n-1\) si ha \(\vect{v}_{j}(\theta-\lambda\Id)=\vect{v}_{j+1}\) e, pertanto, \(\vect{v}_{j}\theta=\vect{v}_{j}\lambda+\vect{v}_{j+1}\text{.}\) Ciò permette di determinare i coefficienti della riga \(j\)-esima. Per determinare l'ultima riga della matrice notiamo che \(\vect{v}_{n}(\theta-\lambda\Id)=\vect{v}(\theta-\lambda\Id)^{n-1}(\theta-\lambda\Id)= \vect{v}(\theta-\lambda\Id)^{n}=\vect{0}\text{.}\) Pertanto \(\vect{v}_{n}\theta=\vect{v}_{n}\lambda\) da cui ricaviamo i coefficienti dell'ultima riga della matrice.

La matrice \(J_{n}(\lambda)\) è triangolare, quindi si calcola immediatamente il suo polinomio caratteristico \((x-\lambda)^{n}\) e, pertanto, l'unico autovalore è \(\lambda\) di molteplicità algebrica \(n\text{.}\) La molteplicità geometrica di \(\lambda\) è uguale a \(n-\rk(\lambda\Idmatrix-J_{n}(\lambda))\text{.}\) Ora

\begin{equation*} \lambda\Idmatrix-J_{n}(\lambda)= \begin{pmatrix} 0 \amp -1 \amp 0 \amp \dots \amp \dots \amp 0\\ 0 \amp 0 \amp -1 \amp \dots \amp \dots \amp 0\\ \vdots \amp \vdots \amp \ddots \amp \ddots \amp \dots \amp \vdots\\ 0 \amp \dots \amp \dots \amp 0 \amp -1 \amp 0 \\ 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp -1\\ 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 0 \end{pmatrix} \end{equation*}

ha, ovviamente, rango \(n-1\) e, dunque, la molteplicità geometrica di \(\lambda\) è uguale a \(1\text{.}\)

Partendo dalla forma canonica primaria e applicando blocco a blocco la scelta di base data dalla proposizione precedente, otteniamo il

Teorema 27.4.2.

Sia \(V\) uno spazio vettoriale di dimensione \(n\) su un campo \(K\) e sia \(\theta\) un endomorfismo di \(V\text{,}\) il cui polinomio caratteristico sia totalmente riducibile (ciò avviene sempre se \(K\) è algebricamente chiuso). Esiste allora una base di \(V\) rispetto a cui \(\theta\) si rappresenta con una matrice diagonale a blocchi del tipo

\begin{equation*} \begin{pmatrix} J_{n_{1}}(\lambda_{1}) \amp 0 \amp \dots \amp 0 \\ 0 \amp J_{n_{2}}(\lambda_{2}) \amp \dots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp \dots \amp J_{n_{s}}(\lambda_{s}) \end{pmatrix} \end{equation*}

dove ciascun \(J_{n_{i}}(\lambda_{i})\) è un blocco di Jordan. La matrice \(J\text{,}\) detta matrice di Jordan, è univocamente determinata a meno dell'ordinamento in cui sono riportati i blocchi.

Anche questo risultato ha un corrispettivo matriciale:

Teorema 27.4.3.

Sia \(A\) una matrice quadrata di ordine \(n\) a coefficienti in un campo \(K\text{.}\) Se il polinomio caratteristico di \(A\) è totalmente riducibile (ciò avviene sempre se \(K\) è algebricamente chiuso) allora \(A\) è simile a una matrice diagonale a blocchi del tipo

\begin{equation*} J\coloneqq \begin{pmatrix} J_{n_{1}}(\lambda_{1}) \amp 0 \amp \dots \amp 0 \\ 0 \amp J_{n_{2}}(\lambda_{2}) \amp \dots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp \dots \amp J_{n_{s}}(\lambda_{s}) \end{pmatrix} \end{equation*}

dove ciascun \(J_{n_{i}}(\lambda_{i})\) è un blocco di Jordan. Inoltre il polinomio caratteristico di \(A\) è \((x-\lambda_{1})^{n_{1}}\cdots (x-\lambda_{s})^{n_{s}}\text{.}\) La matrice \(J\) è univocamente determinata a meno dell'ordinamento in cui sono riportati i blocchi e viene detta forma canonica di Jordan della matrice \(A\text{.}\)

Nota 27.4.4.

Anche in questo caso sarebbe più corretto parlare di forma normale di Jordan.

Osservazione 27.4.5.

Notiamo che nella matrice di Jordan data dai teoremi precedenti non è detto che gli autovalori \(\lambda_{1}\text{,}\) \(\dots\text{,}\) \(\lambda_{s}\) siano tutti diversi.

Corollario 27.4.6.

Due matrici quadrate \(A\) e \(B\) dello stesso ordine aventi polinomio caratteristico totalmente riducibile sono simili se e solo se hanno la stessa forma canonica di Jordan (a meno eventualmente dell'ordine dei blocchi).

Dimostrazione.

Sia \(J\) la forma canonica di Jordan di \(A\) (cioè sia \(A\) simile a \(J\)). Se \(B\) è simile ad \(A\) allora è simile anche a \(J\text{.}\) Viceversa se \(B\) ha \(J\) come sua forma canonica di Jordan (cioè se \(B\) è simile a \(J\)) allora \(A\) e \(B\text{,}\) essendo simili a \(J\) sono simili fra loro.

Osservazione 27.4.7.

Nel caso in cui i blocchi di Jordan abbiano tutti ordine \(1\) otteniamo una usuale matrice diagonale.

Data una matrice \(A\) il cui polinomio caratteristico è totalmente riducibile, viene allora naturale chiedersi come sia possibile determinare una matrice di Jordan a essa simile. Uno dei metodi per far questo è quello di cercare degli invarianti per similitudine. Più precisamente, data una matrice quadrata \(A\text{,}\) ci chiediamo che cosa hanno in comune le matrici simili ad \(A\text{.}\) Sappiamo già, ad esempio, che matrici simili hanno lo stesso determinante, lo stesso rango e lo stesso polinomio caratteristico. Diamo allora il

Lemma 27.4.8.

Se \(A\) e \(B\) sono matrici quadrate simili tra loro allora le matrici \((A-k\Idmatrix)^{t}\) e \((B-k\Idmatrix)^{t}\) sono simili fra loro per ogni \(k\in K\) e ogni \(t>0\text{.}\)

Dimostrazione.

Esiste una matrice invertibile \(M\) tale che \(M^{-1}AM=B\text{.}\) Ma allora \(M^{-1}(A-k\Idmatrix)M=M^{-1}AM-kM^{-1}\Idmatrix M=B-k\Idmatrix\text{.}\) Dunque l'enunciato è vero per \(t=1\text{.}\) Procediamo per induzione su \(t\text{:}\) per \(t\gt 1\) abbiamo allora

\begin{align*} M^{-1}(A-k\Idmatrix)^{t}M\amp =M^{-1}(A-k\Idmatrix)^{t-1}(A-k\Idmatrix)M\\ \amp =M^{-1}(A-k\Idmatrix)^{t-1}MM^{-1}(A-k\Idmatrix)M\\ \amp =(B-k\Idmatrix)^{t-1}(B-k\Idmatrix)=(B-k\Idmatrix)^{t} \end{align*}

Lemma 27.4.9.

Dato un blocco di Jordan \(J_{n}(\lambda)\) si ha \(\rk(J_{n}(\lambda)-\lambda\Idmatrix)^{t}=n-t\) per \(t\le n\) e \(\rk(J_{n}(\lambda)-\lambda\Idmatrix)^{t}=0\) per \(t>n\text{.}\)

Dimostrazione.

La matrice \(J_{n}(\lambda)-\lambda\Idmatrix\) è la matrice

\begin{equation*} \begin{pmatrix} 0 \amp 1 \amp 0 \amp \dots \amp \dots \amp 0\\ 0 \amp 0 \amp 1 \amp \dots \amp \dots \amp 0\\ \vdots \amp \vdots \amp \ddots \amp \ddots \amp \dots \amp \vdots\\ 0 \amp \dots \amp \dots \amp 0 \amp 1 \amp 0 \\ 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 1\\ 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 0 \end{pmatrix} \end{equation*}

il cui rango è \(n-1\text{.}\) Calcolando le potenze successive di \(J_{n}(\lambda)-\lambda\Idmatrix\) si trovano le matrici

\begin{equation*} \begin{pmatrix} 0 \amp 0 \amp 1 \amp 0 \amp \dots \amp \dots \amp 0\\ 0 \amp 0 \amp 0 \amp 1 \amp \dots \amp \dots \amp 0\\ \vdots \amp \vdots \amp \vdots \amp \ddots \amp \ddots \amp \dots \amp \vdots\\ 0 \amp 0 \amp \dots \amp \dots \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 1\\ 0 \amp 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 0\\ 0 \amp 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 0 \end{pmatrix}\qquad \begin{pmatrix} 0 \amp0 \amp 0 \amp 1 \amp 0 \amp \dots \amp \dots \amp 0\\ 0 \amp0 \amp 0 \amp 0 \amp 1 \amp \dots \amp \dots \amp 0\\ \vdots \amp\vdots \amp \vdots \amp \vdots \amp \ddots \amp \ddots \amp \dots \amp \vdots\\ 0 \amp0 \amp 0 \amp \dots \amp \dots \amp 0 \amp 1 \amp 0 \\ 0 \amp0 \amp 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 1\\ 0 \amp0 \amp 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 0\\ 0 \amp0 \amp 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 0\\ 0 \amp0 \amp 0 \amp \dots \amp \dots \amp 0 \amp 0 \amp 0 \end{pmatrix} \end{equation*}

e così via. A ogni passo il rango decresce di \(1\) fino ad azzerarsi.

Sia allora \(A\) una matrice quadrata il cui polinomio caratteristico è totalmente riducibile. Sappiamo che \(A\) è simile a una matrice di Jordan

\begin{equation*} J\coloneqq \begin{pmatrix} J_{n_{1}}(\lambda_{1}) \amp 0 \amp \dots \amp 0 \\ 0 \amp J_{n_{2}}(\lambda_{2}) \amp \dots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp \dots \amp J_{n_{s}}(\lambda_{s}) \end{pmatrix} \end{equation*}

dove \(\lambda_{1}\text{,}\) \(\dots\text{,}\) \(\lambda_{s}\) sono gli autovalori (non necessariamente distinti) di \(A\text{.}\) Sia \(\lambda\) uno degli autovalori di \(A\text{.}\) La matrice \(J-\lambda\Idmatrix\) è ovviamente la matrice

\begin{equation*} \begin{pmatrix} J_{n_{1}}(\lambda_{1})-\lambda\Idmatrix \amp 0 \amp \dots \amp 0 \\ 0 \amp J_{n_{2}}(\lambda_{2})-\lambda\Idmatrix \amp \dots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp \dots \amp J_{n_{s}}(\lambda_{s})-\lambda\Idmatrix \end{pmatrix} \end{equation*}

e, per la Proposizione 27.2.2 si ha che \((J-\lambda\Idmatrix)^{t}\) è la matrice

\begin{equation*} \begin{pmatrix} (J_{n_{1}}(\lambda_{1})-\lambda\Idmatrix)^{t} \amp 0 \amp \dots \amp 0 \\ 0 \amp (J_{n_{2}}(\lambda_{2})-\lambda\Idmatrix)^{t} \amp \dots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \vdots\\ 0 \amp 0 \amp \dots \amp (J_{n_{s}}(\lambda_{s})-\lambda\Idmatrix)^{t} \end{pmatrix}\text{.} \end{equation*}

Ora se \(\lambda\) è diverso da \(\lambda_{i}\) il blocco \(J_{n_{i}}(\lambda_{i})-\lambda\Idmatrix\) è una matrice triangolare i cui elementi lungo la diagonale sono tutti uguali a \(\lambda_{i}-\lambda\) e sono dunque non nulli: ciò implica che \(J_{n_{i}}(\lambda_{i})-\lambda\Idmatrix\) è invertibile così come tutte le potenze \((J_{n_{i}}(\lambda_{i})-\lambda\Idmatrix)^{t}\) e, pertanto, questo blocco ha rango \(n_{i}\text{.}\) Se invece \(\lambda\) è uguale a \(\lambda_{i}\text{,}\) grazie al Lemma 27.4.9 sappiamo che il rango di \((J_{n_{i}}(\lambda_{i})-\lambda\Idmatrix)^{t}\) decresce a ogni passo di \(1\) fino ad azzerarsi. Per la Proposizione 27.2.2 sappiamo che il rango di \((J-\lambda\Idmatrix)^{t}\) è uguale alla somma dei ranghi dei blocchi \((J_{n_{i}}(\lambda_{i})-\lambda\Idmatrix)^{t}\text{.}\) Dunque, se \(n\) è l'ordine della matrice \(A\) e consideriamo la sequenza \(n\text{,}\) \(\rk(J-\lambda\Idmatrix)\text{,}\) \(\rk((J-\lambda\Idmatrix)^{2})\text{,}\) \(\dots\text{,}\) abbiamo che al primo passo questa sequenza decresce di un numero uguale al numero di blocchi di Jordan relativi all'autovalore \(\lambda\text{,}\) al secondo passo decresce di un numero uguale al numero di blocchi di Jordan relativi all'autovalore \(\lambda\) di ordine almeno \(2\) e così via. Dunque, la conoscenza dei ranghi delle matrici \((J-\lambda\Idmatrix)^{t}\) ci permette di determinare il numero di blocchi di Jordan relativi a \(\lambda\) e i loro ordini e, quindi, utilizzando lo stesso argomento per ciascun autovalore, di determinare \(J\) stessa. Si potrebbe pensare che ciò non sia di grande aiuto, visto che per calcolare il rango di \((J-\lambda\Idmatrix)^{t}\) dobbiamo conoscere \(J\text{:}\) tuttavia il Lemma 27.4.8 ci dice che \((J-\lambda\Idmatrix)^{t}\) e \((A-\lambda\Idmatrix)^{t}\) sono simili per ogni \(t\) e, in particolare, hanno lo stesso rango. Possiamo allora delineare un procedimento per determinare \(J\) a partire da \(A\text{.}\)

Algoritmo 27.4.10.

Sia \(A\) una matrice quadrata di ordine \(n\text{.}\) Vogliamo determinare, se esiste, una matrice di Jordan a essa simile.

  • Calcoliamo il polinomio caratteristico \(\det(x\Idmatrix-A)\) di \(A\text{.}\) Se il polinomio caratteristico non è totalmente riducibile allora \(A\) non è simile a una matrice di Jordan e non possiamo proseguire oltre. Se invece il polinomio caratteristico è totalmente riducibile lo scriviamo nella forma \((x-\lambda_{1})^{d_{1}}\cdots(x-\lambda_{s})^{d_{s}}\) dove \(\lambda_{1}\text{,}\) \(\dots\text{,}\) \(\lambda_{s}\) sono gli autovalori distinti di \(A\) e \(d_{1}\text{,}\) \(\dots\text{,}\) \(d_{s}\) sono le corrispondenti molteplicità algebriche.

  • Per ciascun autovalore \(\lambda_{i}\) consideriamo la sequenza di numeri \(n\text{,}\) \(\rk(A-\lambda_{i}\Idmatrix)\text{,}\) \(\rk((A-\lambda_{i}\Idmatrix)^{2})\text{,}\) \(\dots\) (si noti \(n\) come primo termine). Questi numeri formano una successione che decresce strettamente fin quando non raggiunge \(n-d_{i}\text{.}\)

  • Il numero \(n-\rk(A-\lambda_{i}\Idmatrix)\) dà il numero di blocchi di Jordan relativi a \(\lambda_{i}\text{,}\) il numero \(\rk(A-\lambda_{i}\Idmatrix)-\rk((A-\lambda_{i}\Idmatrix)^{2})\) dà il numero di blocchi di Jordan relativi a \(\lambda_{i}\) di ordine almeno \(2\) e così via. Determiniamo così il numero di blocchi di Jordan relativi a \(\lambda_{i}\) e i loro ordini.

Osservazione 27.4.11.

Il procedimento precedente ci dice come determinare una matrice di Jordan \(J\) simile a una matrice quadrata \(A\) assegnata, ma non ci dice come determinare una matrice \(M\) invertibile tale che \(M^{-1}AM=J\text{:}\) ovviamente esiste un metodo anche per determinare una tale matrice \(M\) ma non lo trattiamo qui.

Esempio 27.4.12.

Si consideri la matrice

\begin{equation*} A\coloneqq \begin{pmatrix} 2\amp0\amp0\amp0\amp0\amp0\amp0\amp0\\ 0\amp2\amp0\amp0\amp0\amp0\amp0\amp0\\ 0\amp3\amp1\amp0\amp1\amp0\amp0\amp0\\ 4\amp0\amp0\amp2\amp0\amp1\amp0\amp1\\ 0\amp0\amp0\amp0\amp1\amp2\amp0\amp0\\ 0\amp0\amp0\amp0\amp0\amp1\amp0\amp1\\ 0\amp0\amp0\amp0\amp0\amp0\amp2\amp0\\ 0\amp0\amp0\amp0\amp0\amp0\amp1\amp2 \end{pmatrix} \end{equation*}

È facile calcolare il polinomio caratteristico di questa matrice (utilizzando a ogni passo opportuni sviluppi secondo Lagrange): si ottiene \(\det(x\Idmatrix-A)=(x-1)^{3}(x-2)^{5}\text{.}\) Gli autovalori sono quindi \(1\text{,}\) di molteplicità algebrica \(3\) e \(2\) di molteplicità algebrica \(5\text{.}\) Si noti che in generale non è detto che gli autovalori di una matrice siano gli elementi della diagonale. Avremo dunque blocchi di Jordan relativi a \(1\) di ordine totale \(3\) e blocchi di Jordan relativi a \(2\) di ordine totale \(5\text{.}\)

Consideriamo l'autovalore \(1\text{:}\) costruiamo la sequenza dei ranghi associata a \(1\text{:}\) il primo numero è \(8\text{,}\) il secondo è \(\rk(A-\Idmatrix)=7\text{,}\) il terzo è \(\rk((A-\Idmatrix)^{2})=6\text{,}\) il quarto è \(\rk((A-\Idmatrix)^{3})=5\) e qui ci fermiamo perché \(5=8-3\) (ovviamente qui non sono riportati, per brevità, i calcoli necessari). Ora disegniamo il grafico come in precedenza:

Abbiamo una prima riga lunga \(1=8-7\text{,}\) una seconda riga lunga \(1=7-6\) e una terza riga lunga pure \(1=6-5\text{.}\) Dunque un'unica colonna alta \(3\text{:}\) c'è un unico blocco di Jordan relativo a \(1\) di ordine \(3\text{.}\) Notiamo però che avremmo potuto risparmiarci qualche calcolo: dopo aver calcolato il rango di \(A-\Idmatrix\) avremmo potuto notare che la riga inferiore del nostro grafico contiene un unico elemento e, pertanto, che le righe successive avrebbero dovuto avere lunghezza \(1\text{.}\)

Passiamo ora all'autovalore \(2\text{.}\) Anche qui diamo, senza esplicitare i calcoli, la sequenza dei ranghi associati: otteniamo \(8\text{,}\) \(5\text{,}\) \(4\text{,}\) \(3\text{,}\) e qui ci fermiamo perché \(3=8-5\text{.}\) Il grafico in questo caso è il seguente:

Abbiamo una prima riga lunga \(3=8-5\text{,}\) una seconda riga lunga \(1=5-4\) e una terza riga lunga pure \(1=4-3\text{.}\) Anche qui, una volta visto che la seconda riga ha lunghezza \(1\) avremmo potuto desumere che la terza riga ha pure lunghezza \(1\text{:}\) si noti inoltre che il numero totale di quadretti deve essere uguale a \(5\text{.}\) Guardando le colonne possiamo ricavare che ci deve essere un blocco di ordine \(3\) e \(2\) blocchi di ordine \(1\text{.}\)

Riassumendo, la matrice \(A\) è simile alla matrice di Jordan:

\begin{equation*} J\coloneqq \begin{pmatrix} 1\amp1\amp0\amp0\amp0\amp0\amp0\amp0\\ 0\amp1\amp1\amp0\amp0\amp0\amp0\amp0\\ 0\amp0\amp1\amp0\amp0\amp0\amp0\amp0\\ 0\amp0\amp0\amp2\amp1\amp0\amp0\amp0\\ 0\amp0\amp0\amp0\amp2\amp1\amp0\amp0\\ 0\amp0\amp0\amp0\amp0\amp2\amp0\amp0\\ 0\amp0\amp0\amp0\amp0\amp0\amp2\amp0\\ 0\amp0\amp0\amp0\amp0\amp0\amp0\amp2 \end{pmatrix}\text{.} \end{equation*}

Ovviamente i blocchi possono essere disposti in qualsiasi ordine. Ad esempio anche la matrice

\begin{equation*} \begin{pmatrix} 2\amp0\amp0\amp0\amp0\amp0\amp0\amp0\\ 0\amp1\amp1\amp0\amp0\amp0\amp0\amp0\\ 0\amp0\amp1\amp1\amp0\amp0\amp0\amp0\\ 0\amp0\amp0\amp1\amp0\amp0\amp0\amp0\\ 0\amp0\amp0\amp0\amp2\amp0\amp0\amp0\\ 0\amp0\amp0\amp0\amp0\amp2\amp1\amp0\\ 0\amp0\amp0\amp0\amp0\amp0\amp2\amp1\\ 0\amp0\amp0\amp0\amp0\amp0\amp0\amp2 \end{pmatrix} \end{equation*}

è simile alla matrice \(A\text{.}\)

Supponiamo ora di voler trovare la forma canonica primaria di \(A\text{.}\) Ricordiamo che abbiamo ottenuto la forma canonica di Jordan a partire da una tale forma agendo blocco per blocco: precisamente il blocco matrice compagna del polinomio \((x-\lambda)^{n}\) è stato sostituito a un blocco di Jordan relativo all'autovalore \(\lambda\text{.}\) Consideriamo allora \(J\text{.}\) Dobbiamo sostituire al blocco di Jordan di ordine \(3\) relativo a \(1\) la matrice compagna del polinomio \((x-1)^{3}\text{,}\) che espanso dà \(x^{3}-3x^{2}+3x-1\text{:}\) la matrice compagna di questo polinomio è

\begin{equation*} \begin{pmatrix} 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ 1 \amp -3 \amp 3 \end{pmatrix}\text{.} \end{equation*}

Si noti che gli elementi dell'ultima riga sono i coefficienti cambiati di segno del polinomio a partire dal termine noto e di grado crescente (tranne il coefficiente direttivo).

Al blocco di Jordan di ordine \(3\) relativo a \(2\) sostituiamo la matrice compagna del polinomio \((x-2)^{3}=x^{3}-2x^{2}+4x-8\) cioè:

\begin{equation*} \begin{pmatrix} 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ 8 \amp -4 \amp 2 \end{pmatrix}\text{.} \end{equation*}

Infine i due blocchi di Jordan di ordine \(1\) sono anche matrici compagne di polinomi di primo grado e non necessitano quindi di modifica. Abbiamo così la matrice:

\begin{equation*} C\coloneqq \begin{pmatrix} 0\amp1\amp0\amp0\amp0\amp0\amp0\amp0\\ 0\amp0\amp1\amp0\amp0\amp0\amp0\amp0\\ 1\amp-3\amp3\amp0\amp0\amp0\amp0\amp0\\ 0\amp0\amp0\amp0\amp1\amp0\amp0\amp0\\ 0\amp0\amp0\amp0\amp0\amp1\amp0\amp0\\ 0\amp0\amp0\amp8\amp-4\amp2\amp0\amp0\\ 0\amp0\amp0\amp0\amp0\amp0\amp2\amp0\\ 0\amp0\amp0\amp0\amp0\amp0\amp0\amp2 \end{pmatrix}\text{.} \end{equation*}

2.

Si consideri la matrice \(A\coloneqq \begin{psmallmatrix} 1 \amp 2 \amp 3 \\ 0 \amp -1 \amp 4 \\ 0 \amp 0 \amp 3 \end{psmallmatrix}\) dove i coefficienti sono elementi di un campo \(F\text{.}\)

(a)

Determinare una matrice di Jordan simile ad \(A\) nel caso in cui \(F=\Razionali\text{.}\)

Soluzione.

La matrice \(A\) è triangolare, quindi i suoi autovalori sono \(1\text{,}\) \(-1\) e \(3\) qualunque sia il campo \(F\) e il polinomio caratteristico di \(A\) è sempre totalmente riducibile. Poiché \(F=\Razionali\text{,}\) gli autovalori sono distinti e hanno tutti molteplicità algebrica \(1\text{.}\) Per ciascun autovalore esiste dunque un unico blocco di Jordan ad esso relativo e tale blocco ha ordine \(1\text{.}\) In altri termini, \(A\) è diagonalizzabile e simile alla matrice \(\begin{psmallmatrix} 1 \amp 0 \amp 0 \\ 0 \amp -1 \amp 0 \\ 0 \amp 0 \amp 3 \end{psmallmatrix}\text{.}\)

(b)

Determinare una matrice di Jordan simile ad \(A\) nel caso in cui \(F=\Interi/p\) con \(p\) numero primo.

Soluzione.

Possiamo riprendere i calcoli del punto precedente. Poiché \(F=\Interi/p\) dobbiamo stabilire, in dipendenza da \(p\text{,}\) se gli autovalori sono distinti o meno. Gli autovalori \(1\) e \(-1\) coincidono se e solo se la loro differenza \(1-(-1)\) è multipla di \(p\text{,}\) cioè se e solo se \(p=2\text{.}\) Analogamente \(1\) e \(3\) coincidono se e solo se \(3-1\) è multiplo di \(p\text{,}\) il che avviene se e solo se \(p=2\text{.}\) Infine \(-1\) e \(3\) coincidono se e solo se \(3-(-1)\) è multiplo di \(2\) il che avviene, ancora una volta, se e solo se \(p=2\text{.}\) Riassumendo: se \(p\ne2\) la matrice \(A\) ha tre autovalori aventi molteplicità algebrica \(1\) e, quindi, come nel caso \(F=\Razionali\) la matrice è simile alla matrice diagonale \(\begin{psmallmatrix} 1 \amp 0 \amp 0 \\ 0 \amp -1 \amp 0 \\ 0 \amp 0 \amp 3 \end{psmallmatrix}\text{.}\) Se, invece \(p=2\text{,}\) la matrice \(A\) ha un unico autovalore \(1\) di molteplicità algebrica \(3\text{.}\) Consideriamo la solita successione dei ranghi partendo da \(3\text{,}\) ordine di \(A\text{.}\) Consideriamo ora la matrice \(A-\Idmatrix= \begin{psmallmatrix} 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{psmallmatrix}\) (si noti che in questo campo \(2=0\)). La matrice \(A-\Idmatrix\) ha rango \(1\text{:}\) dall'intero \(3\) siamo scesi di \(2\) e, quindi ci sono due blocchi relativi a \(1\text{.}\) Non abbiamo bisogno di proseguire oltre: ci deve necessariamente essere un blocco di ordine \(2\) e un blocco di ordine \(1\) e, dunque, \(A\) è simile alla matrice \(\begin{psmallmatrix} 1 \amp 1 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{psmallmatrix}\text{.}\)

Gruppo di esercizi.

Per ciascuna delle seguenti coppie di matrici a coefficienti in \(\Razionali\) stabilire se sono simili tra loro.

3.

\(A\coloneqq \begin{psmallmatrix} 3 \amp 0 \amp 2 \\ 0 \amp 3 \amp 1 \\ 0 \amp 0 \amp 1 \end{psmallmatrix}\) e \(B\coloneqq \begin{psmallmatrix} 1 \amp 4 \amp 0 \\ 1 \amp 3 \amp 2 \\ 1 \amp -2 \amp 3 \end{psmallmatrix}\text{.}\)

Soluzione.

La matrice \(A\) è triangolare, quindi i suoi autovalori sono gli elementi lungo la diagonale. Dunque, \(A\) ha come autovalori \(3\) di molteplicità algebrica \(2\) e \(1\) di molteplicità algebrica \(1\text{.}\) Il polinomio caratteristico di \(A\) è \((x-3)^{2}(x-1)\text{.}\)

Il polinomio caratteristico di \(B\) è \(\det(x\Idmatrix-A)\) cioè \(\begin{vsmallmatrix} x-1 \amp -4 \amp 0 \\ -1 \amp x-3 \amp -2 \\ -1 \amp 2 \amp x-3 \end{vsmallmatrix}=x^{3}-7x^{2}+15x-9\text{.}\) Espandendo il polinomio caratteristico di \(A\) troviamo lo stesso polinomio: dunque, \(B\) e \(A\) hanno lo stesso polinomio caratteristico e questo è totalmente riducibile. Calcoliamo la forma canonica di Jordan per \(A\) e \(B\text{.}\) Poiché \(1\) ha molteplicità algebrica \(1\text{,}\) per entrambe le matrici c'è un unico blocco di Jordan di ordine relativo a \(1\text{.}\) Consideriamo l'autovalore \(3\) e calcoliamo la successione dei ranghi associati a \(3\text{,}\) partendo, come sempre, dall'ordine della matrice, cioè \(3\text{.}\) La matrice \(A-3\Idmatrix\text{,}\) cioè la matrice \(\begin{psmallmatrix} 0 \amp 0 \amp 2 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp -2 \end{psmallmatrix}\text{,}\) ha rango \(1\text{.}\) Il rango è sceso di \(3-1=2\text{:}\) avremo, quindi, \(2\) blocchi di Jordan relativi a \(3\text{.}\) Consideriamo ora la successione dei ranghi per la matrice \(B\text{:}\) La matrice \(B-3\Id\text{,}\) cioè la matrice \(\begin{psmallmatrix} -2 \amp 4 \amp 0 \\ 1 \amp 0 \amp 2 \\ 1 \amp -2 \amp 0 \end{psmallmatrix}\text{,}\) ha rango \(2\text{.}\) Il rango è sceso di \(3-2=1\text{:}\) avremo, quindi, un unico blocco di Jordan relativo a \(3\text{.}\) Le due matrici non sono dunque simili.

4.

\(A\coloneqq \begin{psmallmatrix} 1 \amp 3 \amp 1 \\ 1 \amp 2 \amp 1 \\ 0 \amp 0 \amp 1 \end{psmallmatrix}\) e \(B\coloneqq \begin{psmallmatrix} 1 \amp 1 \amp 0 \\ 1 \amp 2 \amp 1 \\ 0 \amp 1 \amp 1 \end{psmallmatrix}\text{.}\)

Soluzione.

La matrice \(A\) ha polinomio caratteristico \((x-1)(x^{2}-3x-1)\text{.}\) La matrice \(B\) ha polinomio caratteristico \(x^{3}-4x^{2}+3x\text{.}\) Espandendo il polinomio caratteristico di \(A\) troviamo il polinomio \(x^{3}-4x^{2}+2x+1\text{.}\) Le matrici hanno polinomio caratteristico differente, quindi non sono simili. Non è necessario calcolare gli autovalori delle due matrici né, tantomeno, la loro (eventuale) forma canonica di Jordan.

5.

\(A\coloneqq \begin{psmallmatrix} 2 \amp -1 \amp 1 \\ 1 \amp 0 \amp 2 \\ 0 \amp 0 \amp 1 \end{psmallmatrix}\) e \(B\coloneqq \begin{psmallmatrix} -1 \amp -2 \amp 2 \\ 1 \amp 2 \amp -1 \\ 0 \amp 1 \amp 2 \end{psmallmatrix}\text{.}\)

Soluzione.

La matrice \(A\) ha polinomio caratteristico \((x-1)(x^{2}-2x+1)\) e la matrice \(B\) ha polinomio caratteristico \(x^{3}-3x^{2}+3x-1\text{.}\) Espandendo il polinomio caratteristico di \(A\) troviamo lo stesso polinomio. Fattorizzando questo polinomio, troviamo \((x-1)^{3}\text{,}\) che è totalmente riducibile e, quindi, possiamo determinare la forma canonica di Jordan per entrambe le matrici. Entrambe hanno \(1\) come unico autovalore di molteplicità algebrica \(3\text{.}\) Consideriamo allora le successioni dei ranghi associate a questo autovalore, partendo dall'ordine delle matrici, cioè \(3\text{.}\) La matrice \(A-\Idmatrix\) cioè la matrice \(\begin{psmallmatrix} 1 \amp -1 \amp 1 \\ 1 \amp -1 \amp 2 \\ 0 \amp 0 \amp 0 \end{psmallmatrix}\) ha rango \(2\text{.}\) Analogamente la matrice \(B-\Idmatrix\) cioè la matrice \(\begin{psmallmatrix} -2 \amp -2 \amp 2 \\ 1 \amp 1 \amp -1 \\ 0 \amp 1 \amp 1 \end{psmallmatrix}\) ha rango \(2\text{.}\) In entrambi i casi siamo scesi di \(3-2=1\text{:}\) pertanto sia la forma canonica di Jordan di \(A\) che la forma canonica di Jordan di \(B\) contengono un unico blocco di Jordan relativo a \(1\) di ordine \(3\text{.}\) Le due matrici \(A\) e \(B\) sono entrambe simili alla matrice \(\begin{psmallmatrix} 1 \amp 1 \amp 0 \\ 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 1 \end{psmallmatrix}\) e sono, pertanto, simili tra loro.
